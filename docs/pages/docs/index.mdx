# Getting Started

## Overview

The Solana Record Service (SRS) is a public good program that allows you to create, update, and delete records for names and TLDs.

## Quick Start

## Manual Installation

To manually add the SRS library to your project, you can use the following commands:

### Typescript / Javascript

```bash [npm]
npm i srs-lib @metaplex-foundation/umi @solana/kit @solana/signers 
```

```bash [yarn]
yarn add srs-lib @metaplex-foundation/umi @solana/kit @solana/signers 
```

```bash [pnpm]
pnpm i srs-lib @metaplex-foundation/umi @solana/kit @solana/signers 
```

### Rust

```bash
cargo add solana-record-service-client
```
