# Getting Started

## Overview

The Solana Record Service (SRS) is a public good program that allows you to create, update, and delete records for names and TLDs.

## Quick Start

## Installation

Install the SRS library within your project:

### Typescript / Javascript

```bash [npm]
npm i srs-lib
```

```bash [yarn]
yarn add srs-lib
```

```bash [pnpm]
pnpm i srs-lib
```

### Rust

```bash
cargo add solana-record-service-client
```
