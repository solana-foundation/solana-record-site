## Installation

Install the SRS library within your project:

### Typescript / Javascript

```bash [npm]
npm i srs-lib
```

```bash [yarn]
yarn add srs-lib
```

```bash [pnpm]
pnpm i srs-lib
```

### Rust

```bash
cargo add solana-record-service-client
```
