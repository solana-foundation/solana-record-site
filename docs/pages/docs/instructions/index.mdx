# Instructions

This section documents all available instructions in the Solana Record System SDK.

## Record Instructions

| Instruction | Description |
|-------------|-------------|
| [Create Record](./create-record) | Creates a new record under a class |
| [Update Record](./update-record) | Updates an existing record's data |
| [Delete Record](./delete-record) | Permanently deletes a record |
| [Freeze Record](./freeze-record) | Freezes or thaws a record |
| [Transfer Record](./transfer-record) | Transfers ownership of a record |

## Tokenized Record Instructions

| Instruction | Description |
|-------------|-------------|
| [Mint Tokenized Record](./mint-tokenized-record) | Mints a token for an existing record |
| [Update Tokenized Record](./update-tokenized-record) | Updates a tokenized record's data |
| [Freeze Tokenized Record](./freeze-tokenized-record) | Freezes or thaws a tokenized record |
| [Transfer Tokenized Record](./transfer-tokenized-record) | Transfers a tokenized record |
| [Burn Tokenized Record](./burn-tokenized-record) | Burns a tokenized record |

## Class Instructions

| Instruction | Description |
|-------------|-------------|
| [Create Class](./create-class) | Creates a new class |
| [Update Class Metadata](./update-class-metadata) | Updates a class's metadata |

## Common Parameters

Most instructions share common parameters:

- `payer`: The account that will pay for the transaction
- `authority`: The account that has control over the record/class
- `options?`: Optional RPC options for the transaction
