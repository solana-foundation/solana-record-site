# Instructions

This section documents all available instructions in the Solana Record System SDK.

## Record Instructions

| Instruction | Description |
|-------------|-------------|
| [Create Record](./create-record) | Creates a new record under a class |
| [Update Record](./update-record) | Updates an existing record's data |
| [Burn Record](./burn-record) | Permanently deletes a record |
| [Burn Tokenized Record](./burn-tokenized-record) | Burns a tokenized record and its associated token |

## Class Instructions

| Instruction | Description |
|-------------|-------------|
| [Create Class](./create-class) | Creates a new class |
| [Update Class](./update-class) | Updates an existing class's metadata |
| [Burn Class](./burn-class) | Permanently deletes a class |

## Common Parameters

Most instructions share common parameters:

- `payer`: The account that will pay for the transaction
- `authority`: The account that has control over the record/class
- `options?`: Optional RPC options for the transaction

## Usage Example

```typescript
import { getCreateRecordInstruction } from "srs-lib";

// Create a record instruction
const instruction = getCreateRecordInstruction({
    payer: payer,
    authority: authority,
    class: classPublicKey,
    // ... other parameters
});
```
